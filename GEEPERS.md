## Vision & Safety

- [Geepers Vision](https://github.com/lukeslp/kernel/tree/master/geepers/skills/source/vision) - Enterprise-grade Computer Vision MCP server featuring OSHA safety evaluations, engineering schematic analysis, and WCAG-compliant alt text generation. Supports OpenAI GPT-4o and xAI Grok-2 Vision.

## Product Management

- [Geepers Product](https://github.com/lukeslp/kernel/tree/master/geepers/skills/source/product) - Product Manager MCP server that reverse-engineers code into PRDs, generates Go-To-Market strategies, and creates technical engineering handoffs for agent swarms.
