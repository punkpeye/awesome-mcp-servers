# Packrift MCP Listing

Packrift publishes a production Model Context Protocol server for packaging and shipping-supplies workflows.

- Storefront: https://packrift.com
- Agent guide: https://packrift.com/pages/agents
- MCP endpoint: https://mcp.packrift.com/mcp
- MCP server card: https://packrift.com/.well-known/mcp/server-card.json
- llms.txt: https://packrift.com/llms.txt
- Source repository: https://github.com/Packrift/packrift-mcp

The Packrift MCP server helps AI agents search the catalog, retrieve product details, check live pricing and inventory, recommend packaging by dimensions and use case, estimate shipping, and build Packrift cart URLs.
